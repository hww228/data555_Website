# Personal Website — Youwei Hu

This repository contains the source code for my personal portfolio website built using **R Markdown, HTML, and CSS**. The website highlights my background in biostatistics, research projects, and technical skills in data science.

🔗 **Live Website:**  
https://hww228.github.io/data555_Website/

---

## 📌 Overview

This website serves as a professional portfolio designed to present:

- My academic background in Biostatistics at Emory University  
- Research experience in EEG biomarker analysis and infectious disease modeling  
- Selected projects demonstrating statistical modeling and data analysis  
- Technical skills across programming, statistics, and data visualization  

The site is designed to be **clean, responsive, and accessible on both desktop and mobile devices**.

## 🚀 Featured Projects

### 🧠 EEG Biomarker Dashboard
Interactive Shiny dashboard analyzing resting-state EEG biomarkers for Alzheimer's disease.

### 🦠 Viral Shedding Modeling
Bayesian hierarchical modeling using Stan to integrate viral load and Ct data into a unified framework.

### 📊 Viral Shedding Data Integration
Pipeline for extracting and standardizing longitudinal viral shedding data from published studies using structured schemas and LLM-assisted processing.
